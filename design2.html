Interest

Computer Science is a general term. There are so many different subjects on it. Software development, networking, web development, AI, cyber security, etc. Being a first-year computer science student I have learnt the basics of computer hardware, software development, web development, networking, cyber security. From what I learnt and what I have researched, there are three topics (software development, machine learning, data science) that I am interested in. On this page, I'll explain what it is and why I am interested in it.

Software Development

Software development literally is the process of creating, designing, deploying and supporting software. In the first-year module, I have a module called "Software Development and Application Modelling". I learnt how to develop procedural programs in Python, and object-oriented programs in C# on this module. I got to know the power of programming and I enjoyed the process of thinking how to solve the problem, debugging, and getting my code to work at the end. It is not straightforward but it is satisfying when I manage to solve the problem. This made me interested in software development and I'm thinking to specialise in software development next year.

Data Science

These days, data is a crucial part of a business. Analysing and using data correctly will benefit the company significantly and helps the business to grow. Therefore, there is a high demand for data scientists. In data science, data scientists have to understand the company problem and solve it using data in order to make an impact on the company. Data science is a mixture of computer science and statistics. They are going to collect data, visualise the data, create models, and so on. The popular language used for data science is Python, which I learnt on my first-year module called "Software Development and Application Modelling".Since I enjoyed Python I researched what kind of things I can do with it and met the subject of data science. I think it is a powerful subject and an interesting career to have.

Machine Learning

Same reason to data science I found this subject when I was researching about Python. AI (Artificial Intelligence) is a popular subject nowadays, and machine learning is a subset of AI. Like humans learn things by experience, it is a subject teaching the machine to do a certain task without programming to do so. For instance, if we want to make a machine that can distinguish between an apple and a banana, by feeding a lot of images (data) to a machine, the machine will learn how to distinguish between an apple and a banana using computer science and statistics. I think the concept of teaching a machine to do a task rather than to program it to do the task is really unique. Since it involves a lot of maths besides programming, it is not an easy subject to get into, but it is an intriguing subject to learn.
